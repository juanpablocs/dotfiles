# dotfiles
My dotfiles for ubuntu 16.04

install git
```bash
sudo apt-get update && sudo apt-get install -y git-core
```
clone and install
```bash
git clone https://github.com/juanpablocs/dotfiles.git && bash ./dotfiles/install.sh
```

### What's inside?
- curl
- terminator
- vim plugins
- zsh theme and plugins (git, search)
- meld
- gitk
- ubuntu tweak
- theme flatbulous
